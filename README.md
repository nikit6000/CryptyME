# CryptyME
Encrypt your memes with 2048 bit RSA key.

### Dependencies

* [OpenSSL](https://github.com/openssl/openssl)

### Usage

```
$ ./CrypryMe [args]
Args:
	-e <file_name> - encrypt file.
	-d <file_name> - decrypt file.
	-k <private_key_pem> - set private key.
```
